## 📋 如果MySQL路径不对

### 查找MySQL安装路径

#### 方法1：使用where命令
在命令行中输入：
```cmd
where mysql
```

#### 方法2：手动查找
MySQL通常安装在以下位置：
- `C:\Program Files\MySQL\MySQL Server 8.0\bin\mysql.exe`
- `C:\Program Files (x86)\MySQL\MySQL Server 8.0\bin\mysql.exe`
- `C:\MySQL\MySQL Server 8.0\bin\mysql.exe`

#### 方法3：使用环境变量
如果MySQL已经添加到环境变量，可以直接使用：
```cmd
mysql -h localhost -u root -p1234 internship < "c:\Users\ASUS\Desktop\project\backend\src\main\resources\sql\simple_add_archives.sql"
```

## 📞 常见问题

### Q1: 提示"找不到指定的路径"

**A:** 检查以下几点：
1. 确认MySQL安装路径是否正确
2. 使用 `where mysql` 命令查找MySQL路径
3. 修改命令中的MySQL路径

### Q2: 提示"Access denied for user 'root'@'localhost'"

**A:** 检查以下几点：
1. 确认MySQL密码是否为 `1234`
2. 如果不是，修改命令中的密码
3. 或者询问正确的MySQL密码

### Q3: 提示"Unknown database 'internship'"

**A:** 检查以下几点：
1. 确认数据库名称是否为 `internship`
2. 如果不是，修改命令中的数据库名称
3. 或者先创建数据库

### Q4: 提示"Can't connect to MySQL server"

**A:** 检查以下几点：
1. 确认MySQL服务是否正在运行
2. 打开"服务"管理器，查看MySQL服务状态
3. 如果未运行，启动MySQL服务
