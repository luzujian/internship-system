## 📞 常见问题

### Q1: 执行批处理文件时提示"mysql不是内部或外部命令"

**A:** 需要将MySQL的bin目录添加到系统环境变量，或者使用完整路径：
```cmd
"C:\Program Files\MySQL\MySQL Server 8.0\bin\mysql.exe" -h localhost -u root -p1234 internship < ...
```

### Q2: 执行后看不到数据

**A:** 检查以下几点：
1. 确认执行成功（查看输出信息）
2. 刷新前端页面
3. 检查数据库连接
4. 执行验证SQL查询

### Q3: 预览按钮点击后没有反应

**A:** 检查以下几点：
1. 确认后端服务正在运行
2. 确认前端服务正在运行
3. 检查浏览器控制台是否有错误
4. 检查文件URL是否可访问

### Q4: 下载按钮点击后没有下载

**A:** 检查以下几点：
1. 确认后端服务正在运行
2. 确认文件URL是否可访问
3. 检查浏览器下载设置
4. 查看浏览器控制台是否有错误
